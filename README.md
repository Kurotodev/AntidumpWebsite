# AntiDump - Total FiveM Protection

The most advanced anti-dump system for FiveM. Military-grade encryption, real-time detection, and total protection against unauthorized extractions.

## Features

- **Total Anti-Dump**: Complete protection against script and resource extraction.
- **Real-Time Detection**: 24/7 monitoring that detects and blocks dump attempts automatically.
- **Low Impact**: Optimized to not affect your server performance. 0% lag guaranteed.
- **Multi-Server**: Flexible license to protect multiple (maximum 3) servers with a single control panel.

## Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd AntidumpWeb

# Step 3: Install dependencies
npm i

# Step 4: Start development server
npm run dev
```
